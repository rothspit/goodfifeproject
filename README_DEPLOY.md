# 🚀 簡単デプロイガイド

## ⚡ 1コマンドデプロイ（推奨）

サーバーにSSHアクセスできる場合、以下のコマンドを実行するだけです：

```bash
ssh root@210.131.222.152 'bash -s' < deploy.sh
```

このコマンドは自動的に以下を実行します：
- ✅ 最新コードの取得
- ✅ 環境変数の設定
- ✅ ビルドキャッシュのクリア
- ✅ フロントエンド・バックエンドの再ビルド
- ✅ PM2プロセスの再起動
- ✅ データベースマイグレーション

## 🐛 エラー: "TypeError: Failed to fetch" が出る場合

### 解決方法1: ブラウザキャッシュをクリア
- **Windows/Linux**: `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`

### 解決方法2: 開発者ツールでストレージをクリア
1. `F12` を押して開発者ツールを開く
2. `Application` タブ → `Clear storage` をクリック
3. ページをリロード

### 解決方法3: サーバーを再デプロイ
上記のデプロイコマンドを再実行してください。

## 📞 CTI受注機能の使い方

1. 管理画面にログイン
2. 右上の緑色「**受注**」ボタンをクリック
3. 電話番号を入力して「検索」
4. 顧客情報・履歴を確認
5. キャストを選択（初回/2回目が自動判定）
6. 受注内容を入力して「受注を確定」
